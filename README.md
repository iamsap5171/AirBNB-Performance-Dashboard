🏡 Airbnb Performance Analysis Dashboard

This project is a **data analytics case study** built using **Power BI**, focused on analyzing Airbnb listings and reviews across multiple global cities. It explores trends in listings growth, pricing, ratings, and customer engagement using two datasets: **Listings** and **Reviews**.

---

📌 Project Overview

The goal of this project is to uncover meaningful insights from Airbnb data, such as:

* Growth trends in listings over time
* Distribution of property types
* City-wise market share
* Pricing differences across room types
* Customer engagement through reviews
* Ratings comparison across cities

The dashboard is designed with a clean and consistent visual theme, making it easy to interpret key metrics at a glance.

🧠 Key Insights

* 📈 Listings saw a **major take-off around 2011** and peaked around **2015**
* 📉 A decline occurred between **2016–2018**, with a sharp drop in **2020** (likely due to global factors)
* 🏙️ A few major cities dominate the market share significantly
* 💰 **Hotel rooms are the most expensive**, while **private rooms are the most affordable**
* ⭐ Some cities maintain consistently high ratings, while others lag behind
* 📝 Review distribution highlights user engagement trends and popularity
